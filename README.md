# Blog-App 



Building homepage with react
 - Routing with react-router-dom
 - Login and register page
 - Authentication
 - Checking if logged in
 - Logout functionality
 - UserContext
 - Create new post page
 - Displaying posts from the database
 - Single post page
 - Edit post page
